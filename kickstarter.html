The Ordina JWorks Kickstarter Programme focuses on young graduates that want to give their career a boost with our extensive two-month training.

Curriculum - What you'll learn

The dev environment

You'll learn about your typical development environment. Topics include Git, Maven, IntelliJ, ...

Development

Java 7, 8 & 9 Language Features

Each new Java version includes new and improved features to enhance our day-to-day programming. It is important to stay up-to-date in order to achieve maximum efficiency as a Java developer.

Development

Introduction to Java EE

Gain a basic knowledge on everything that Java EE has to offer.

Development

Introduction to JPA & Hibernate

Learn about the JPA specification for data persistence and the most common implementation: Hibernate.

Development

Clean code deepdive

Writing clean code is what separates good developers from average developers. If you aren't paying attention to how you structure and wire your code, it can rapidly become a burden to maintain and comprehend what a piece of code is doing exactly.

Development

Java Unit Testing & Mocking

Writing good unit tests to test the behaviour of your code is a core skill of any developer nowadays. A new functionality should always come together with a series of unit tests. At the same time, writing good and clear unit tests helps you to document what your functionality is doing exactly.

Development

Spring Framework basics

The Spring Framework is the most popular Java framework and should be well-known by all Java developers. It offers many projects to help you build up your architecture and when trying to come up with a solution for a specific use case such as batch processing, security and data access.

Development

Introduction to web services

An introduction to SOAP (XML) and REST (JSON) web services.

Development

HTML5, CSS3, JavaScript, Sass

This session will cover the new things in the HTML5 spec (elements, API's), the possibilities with CSS3 and how to write structured CSS with Sass.

Development

TypeScript

TypeScript is a typed superset of JavaScript that compiles to plain JavaScript. It allows us to use new features of the ES-spec (classes, interfaces, …), adds strong typing and adds Typings for autocompletion. It will also speed up development and you will write less error prone code.

Development

Introduction to Angular

Angular has been around some time at Ordina. It is the preferred MV* framework within JWorks. We will look into Angular (v2+) and will give you some tips and tricks on how to build a performance-driven enterprise application.

Development

Frontend build tools, testing, package managers etc

The number of possible tools for testing and packaging a frontend application have grown significantly over the last few years. When finished with these courses, you won't be overwhelmed anymore and you'll know which tool to use for your newest project.

Frontend

Ionic 2

Our preferred technology for building mobile applications is the Ionic framework. Ionic and Angular are two peas in a pod. We'll teach you how to build and deploy a mobile application.

Frontend

Security Principles

We believe that every Ordina consultant should have security in his or her DNA. During this course, you'll learn to identify possible security risks as early as possible in a development process.

Development

Git Best Practices

Good knowledge of Git is invaluable. One of the biggest advantages of Git is its branching capabilities. We'll teach you the best practices for working with Git.

Development

MongoDB

MongoDB is a document database with the scalability and flexibility that you want with the querying and indexing that you need. You will learn how to integrate and work with MongoDB by building a small application.

Development

PaaS platforms: PCF & OpenShift

Platform as a service is a category of cloud computing services that allow customers to develop, run, and manage applications without the complexity of building and maintaining the infrastructure. We will take a look at two popular platforms: OpenShift and Pivotal Cloud Foundry.

Development
