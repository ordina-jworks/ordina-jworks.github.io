Als API architect ben je de brug tussen business en IT , je weet hoe je organisaties ondersteunt in de ontwikkeling van hun API strategie. Vanuit jouw ervaring weet je business vragen te vertalen naar de meest optimale IT oplossing. Dit doe je door het applicatielandschap en herbruikbare services van de klant in kaart te brengen. Steeds vertrek je vanuit het service economy en API-first denken. Daarnaast heb je ervaring met integratieprojecten en ken je de verschillende patterns die je kan toepassen om van een organisatie een connected company te maken. Je hebt een visie over hoe je een effectieve API governance kan neerzetten binnen de organisatie met als resultaat een duidelijke API aanpak en referentie-architectuur. Als specialist ben je perfect op de hoogte van het belang van goed ontworpen API’s. Hierbij heb je zelf voldoende technische bagage en expertise als technical architect om teams van developers en analisten op een correcte manier API’s te laten implementeren.

Wie we zoeken?

  • Je hebt 10 jaar relevante werkervaring met minstens 2 jaar als applicatie- of solution architect
  • Je hebt diepgaande kennis van RESTful API design, API-first development en API contracten zoals OpenAPI, Pact of Spring Cloud Contract
  • Je hebt een technische achtergrond in cloud-native development met goed zicht op het Java landschap, containerization en orchestration, en cloud platformen zoals Azure of AWS
  • Je bent vertrouwd met ontwikkelingsmethodologieën, normen, technische risico’s en architectural patterns waaronder event-driven architectures, cloud-native architectures, n-tier, onion architecture, …
  • Je hebt ervaring met application modernization en integration patterns
  • Expertise in API gateway producten zoals Azure API Manager, Amazon API Gateway of een andere is een pluspunt
  • Je deelt je kennis met je collega’s via onze competence centers over onderwerpen zoals architectuur, API design, enterprise integration, cloud en security
  • Je kan communiceren in het Nederlands en Engels, Frans is een pluspunt

Wat we bieden

  • Je draagt bij tot uitdagende projecten bij top internationale en Belgische bedrijven uit diverse sectoren
  • Je werkt in een hightech omgeving met focus op kennis en innovatie, een open no-nonsense bedrijfscultuur met ruimte voor eigen initiatief
  • Je maakt deel uit van JWorks, een dynamische development unit binnen Ordina
  • Stilstaan is achteruitgaan, je komt in een architecten team terecht waar kennis actief gedeeld wordt
  • Je houdt je technische en sociale vaardigheden up-to-date op onze Ordina Academy en wordt ondersteund bij het behalen van certificaten
  • Je krijgt de opportuniteit om deel te nemen aan onze competence center workshops om je kennis up-to-date te houden of uit te breiden
  • Je ontvangt een competitief salarispakket met bijkomende voordelen zoals een bedrijfswagen, laptop, verzekeringspakket, maaltijdcheques, onkostenrekening etc.
  • Je kan deelnemen aan leuke business- en teambuildingsactiviteiten
  • Je kan je nettosalaris optimaliseren dankzij onze IP-reward
